# Portfolio Images

Place your project images in this folder. Keep images under 1MB each for best GitHub performance.

## Recommended sizes:
- **Project A (tall/vertical)**: 1400x1400px minimum
- **Project B (wide/horizontal)**: 1200x800px minimum  
- **Projects C, D, E, F, G, H (square)**: 1000x1000px minimum

## Supported formats:
- JPEG (.jpg, .jpeg) - best for photos
- PNG (.png) - for graphics with transparency
- WebP (.webp) - modern, smaller file sizes

Just drop images here and reference them in `index.html` as:
`src="assets/images/project-a.jpeg"`
